 # 更新日志
 
+## [4.6.0] - 2026-03-28
+
+### 变更
+
+- ✨ **WebUI 个人设置**：新增「调试」开关 `debug_metrics_panel`（按用户写入 PG），启用后在检测控制台侧栏显示会话帧数、FPS、单帧耗时及摄像头/舞台分辨率等指标。
+- 🔧 **WebUI**：保存「个人设置」成功后自动返回检测主界面；账户页个人设置保存成功后跳转 `/webui`。
+- 🔧 **WebUI 设置持久化**：前端保存时提交完整 `WebUISettings` 快照；`POST /api/webui/settings` 使用 `exclude_unset=True`，保留显式 `null`（如 `camera_device_id`）；布尔选项在服务端统一归一化；补充全量设置持久化回归测试。
+
+---
+
+## [4.5.1] - 2026-03-27
+
+### 变更
+
+- 🔧 **WebUI**：将控制台侧栏「参数调整」（判定阈值、推理间隔、JPEG、预览模式、绘制与导出选项等）合并至「个人设置」面板，移除侧栏重复控件与独立「保存参数」按钮；持久化仍通过「保存个人设置」一次完成。
+
+---
+
+## [4.5.0] - 2026-03-27
+
+### 变更
+
+- ✨ **WebUI 个人设置与运行时参数（向后兼容 `/api/webui` 契约）**
+  - 控制台个人设置全屏遮罩，避免切换页卸载摄像头导致黑屏；参数预设（YAML）、推理模型侧重、CPU 档位、导出选项等与后端 `apply_webui_runtime_settings` 对齐
+  - WebUI 参数按**登录用户**写入 PostgreSQL（`users.webui_settings_json`），`GET/POST /api/webui/settings` 同步；`infer`/`start` 前按需从数据库装载，减轻进程重启后设置被默认值覆盖的问题
+- ✨ **导出证据元数据按用户入库**
+  - 新增 `webui_evidence_exports` 表（路径 + 告警/摘要等元数据，不存视频二进制）；事件落盘后写入；`GET /api/webui/exports` 以数据库为权威列表，支持 `limit`/`offset`；管理员可选 `user_id` 筛选
+  - 用户管理页支持按用户加载导出摘要
+- ✨ **账户与历史页面**
+  - 新增 `GET /user`、`GET /user/admin`（`user.html`）；普通用户查看本人历史与共用个人设置；管理员访问 `/user` 时前端跳转 `/user/admin` 以查看全量/按用户筛选；检测控制台工具栏增加「账户与历史」入口
+- 🔧 前端抽取共用 `ExportLinks` 组件；`useVisualWebUI` 支持可选跳过挂载时导出列表请求（账户页独立拉取历史）
+

+"""WebUI 参数预设加载与合并。"""
+from __future__ import annotations
+
+from copy import deepcopy
+from pathlib import Path
+from typing import Any, Dict
+
+import yaml
+
+from src.core.config_loader import ConfigLoader
+
+__all__ = [
+    "apply_preset_to_config",
+    "deep_merge_dict",
+    "load_parameter_presets",
+]
+
+_PRESETS_CACHE: Dict[str, Dict[str, Any]] | None = None
+
+
+def _presets_path() -> Path:
+    return Path(__file__).resolve().parents[2] / "config" / "webui_parameter_presets.yaml"
+
+
+def load_parameter_presets() -> Dict[str, Dict[str, Any]]:
+    global _PRESETS_CACHE
+    if _PRESETS_CACHE is not None:
+        return _PRESETS_CACHE
+    path = _presets_path()
+    if not path.is_file():
+        _PRESETS_CACHE = {}
+        return _PRESETS_CACHE
+    raw = yaml.safe_load(path.read_text(encoding="utf-8")) or {}
+    if not isinstance(raw, dict):
+        _PRESETS_CACHE = {}
+        return _PRESETS_CACHE
+    _PRESETS_CACHE = {str(k): (v if isinstance(v, dict) else {}) for k, v in raw.items()}
+    return _PRESETS_CACHE
+
+
+def deep_merge_dict(base: Dict[str, Any], override: Dict[str, Any]) -> Dict[str, Any]:
+    out = deepcopy(base)
+    for key, val in override.items():
+        if key in out and isinstance(out[key], dict) and isinstance(val, dict):
+            out[key] = deep_merge_dict(out[key], val)
+        else:
+            out[key] = deepcopy(val)
+    return out
+
+
+def apply_preset_to_config(loader: ConfigLoader, preset_name: str) -> bool:
+    """
+    将命名预设合并进 ConfigLoader 内存配置。
+    Returns:
+        是否成功应用（预设存在且已合并）。
+    """
+    presets = load_parameter_presets()
+    preset = presets.get(preset_name)
+    if not preset:
+        return False
+    for section, updates in preset.items():
+        if not isinstance(updates, dict):
+            continue
+        current = loader.get_section(section, {})
+        merged = deep_merge_dict(current, updates)
+        loader.config[section] = merged
+    return True
